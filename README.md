# **Semester Project**

Community Science Museum
![Landing Page](/assets/community%20science%20.png)

## **Description**

Responsive website for Community Science Museum. Showcasing the museums exhibitions, ticket-purchase and information about the museum.
Designed and developed a responsive museum website in a two-person team.
The website includes:

- Landing page - Displays some of the musueum features and allowing the user to navigate to different pages.
- Visit-us page - Displays the location of the museum, opening hours, admission and accessebility. Informes about giftshop and cafè.
- Explore page - Information for kids, teachers and researchers. Useful informations about the clubs and schooltrips the museum offers. Also includes a button navigating to contact-form.
- Exhibitions and Events - Information about the different exhibiutions and events as well as buttons navigating to booking-page.
- Booking Page - Simulating booking of tickets, and displays a contact form.
- Get Involved Page - Information about volunteering, internships and how you can suppert the museum.
- Success Page - Displays success when booking tickets.

## **Getting Started**

### **Installing**

1. Clone the repository:

```bash
 git clone https://github.com/emifos/Semester-Project.git
```

2. Navigate to the project folder

```bash
cd Semester-Project
```

3. Open the project in code ediot (VS Code)

```bash
code .
```

4. Start the project using Live Server

- Open index.html using Live Server Extension
